# 模块文档索引

> 本目录按模块存放功能介绍与进度总结。
> 每次开始新模块前，先在此创建 `<module-name>/README.md` 与 `<module-name>/progress.md`，并同步更新 `docs/TODO.md`。

## 目录命名规范

- 小写英文 + 数字，单词间用 `-` 连接。
- 示例：`shooting-system`、`ui-prefab-workflow`、`luban-config-system`。
